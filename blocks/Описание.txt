	showFigure( needGeneratF = true ) - создает фигуру на панели
	если needGeneratF = false фигура не генерируется случайно или не берется из набора
	и нужно присвоить панели фигуру заранее, а потом вызвать этот метод для отображения фигуры
	в панели
	
	show() - рисует панель, в ней рисует фигуру и 2 кнопки. 1 кнопка - открытие панели
	поворота. 2 - создание новой фигуры на текущей панели. обработчки собития - создает
	случайную фигуру в панели и отнимает манетки.
	
	generationFigure() - генерирует и возвращает случайную фигуру или берет ее из 
	списка с фигурами. список фигур - строка в объекте констант
	
	removeFigure() - удаляет фигуру из панели
	
	при "поднятии" фигуры с панели кнопки поворота 
	и добавлении фигуры становятся не активными

	PanelBonus - обьект создающий панель бонусов
	в панели есть 5 кнопок 1-4 бонусы и кнопка закрытия панели
	если открыта панель 4го бонуса то кнопка закрыть закрывает обе панели
	при нажатии на кнопку в бонуса создается переменная в обьекте Game - текущий выбранный бонус
	обработчик touchBonus принимает evt номер бонуса выбирается evt.target.num
	
	1 бонус - удалении линии   обрабатывается в Game -> touchUp
	2 бонус - удалении столбца обрабатывается в Game -> touchUp
	3 бонус - создает 3 новых фигуры в панелях с фигурами
	4 бонус - создает панель с выором 3 конкретных фигур для замены в панелях
	
	бонусы уменьшают кол-во монет
	
	конструктор получает _parent _x _y _width _heigth

	Game - обьект который управляет логикой игры.
	
	в конструкторе создаются свойтсва:
	this.gameField        = Main.windGame.gameField;
	this.panelsFigures    = Main.windGame.panelsFigures;
	this.touchBlock       = Main.windGame.touchBlock;
	this.panelScore       = Main.windGame.panelScore;
    this.panelPlayer2     = Main.windGame.panelPlayer2;
    this.gameFieldPlayer2 = Main.windGame.gameFieldPlayer2;
	
	они беруться из окна игры
	
	в обьекте есть 3 обрабочика событий мыши
	
	1 touchDown() принимает evt определяет область нажатия
	если нажатие произшло на фигуре она определяется как текущая,
	панель фигуры и фигура и панель переносятся на передний план.
	
	2 touchMove()  принимает evt двигает выделенную фигуру по экрану
	
	3 touchUp() определяет в какие координаты вставить фигуру или 
	если есть выбранный бонус то использует его
	
	проверяте при установке фигуры можно ли вствить 
	фигуру в выбранные координаты методом checkInsertFigure()
	
	после установки 3 фигуры создает 3 новых в панелях с фигурами
	
	проверяет количество удаляемых линий методом checkLinesDel()
	и отправляет их в обьект gameField для удаления
	
	каждую установку фигуры проверяет на окончание игры методом
	checkEndSteps()
	
	после установки обнуляет выбранные фигуру и панель
	
	fillPanelsWithF() - метод ля заполения 
	панелей фигур фигурами из списка для онлайна
	
	checkInsertFigure() - метод, который принимает field, figure, startI, startJ
	он определяет возможность вставить фигуру в текуюие координаты
	возвращает true или false
	
	checkThisLineDel() - метод, принимающий массив с значениями клеток в поле
	и проверяет нудно ли удалить эту линию возвращает true или false
	
	checkLinesDel() - метод, который принимает массив поля и проверят сколько линий
	и столбцов можно удалить из поля
	возвращает массив с массивами в первом из которых кол-во линий
	во втором кол-во столбцов
	
	delLastInsertFigure() - метод, который удаляет последнюю фигуру с поля
	и возвращает последнюю использованную фигуру в ее панель 
	
	checkEndSteps() - метод, который проверяет возможно ли вставить фигуры в поне
	принимает поле игры и поле фигуры fieldGame, fieldsFigures
	
	delLineBon() - метод 1 бонуса, который оправляет 
	номер линии в GameField для удаления
	
	delColBon() - метод 2 бонуса, который оправляет 
	номер столбца в GameField для удаления
	
	recreateFForContinueG() - метод, для продолжения соло игры
	создает 3 новых фигуры с полем 2х2 и очищает 5 и 6 линию на поле
	
	bonusRecreateFigures() - метод, для создания 3 новых фигур 
	для бонуса 3
	
	checkActiveButRotF() - метод, который проверяет тип фигуры
	после установки и делает ее поворот активным если фигура не квадрат
	
	generateStringsFiguresOfChar() - создает строку номеров фигур
	из символов
	
	translationNumToChar() - метод, принимающий номер фигуры
	и создающий символ для нее номера из строки stringForGeneration
	
	generateStringFiguresOfNum() - метол, принимающий строку символов
	и возвращающий строку с символами фигур
	
	selectAct() - метод, принимющий обьект после приема сообщения от сервера
	проверяет какие действия совершил противник, началась ли игра, 
	началась ли игра с ботом
	
	checkWinner() - метод, который проверяет победителя в онлайне
	и открывает окно окончания игры в онлайне
	
	startGame() - метод запускающий игру и таймер в 3 минуты 
	проверяющий по окончаню кто победил
	
	stPlayerExpectation() - метод, который запусткает таймер 
	для отправки сообщения на сервер о поиске противника
	
	setActiveButCancelMove() - метод, который делает
	кнопку отмены хода  активной
	
	setNActiveButCancelMove() - метод, который делает
	кнопку отмены хода не активной

	Bot - объект создающий бота для игры с игроком
	есть 2 массива 10х10 один поле текущее
	другой поле для проверки возможности вставить фигуру
	
	в этом файле есть объект figure в котором поля фигуры,
	номер и ценность ее на поле
	
	startGame() - метод, который начинает игру бота создает 3 фигуры
	и таймер который за тик выбирает какую из 3 фигур лучше поставить
	на поле
	
	stopGame() - метод, который останавливает таймер игры бота
	
	delLine() - метод, который удаляет линии из поля на котором играет бот
	принимает массив с двумя массивами для удаления столбцов и линий
	
	getLengthLine() - метод, который принимает массив-линию со значениями клеток на поле
	возвращает количество заполненых клеток
	
	checkBestCoords() - метод, который принимает фигуру, i, j
	проверяет координаты для фигуры и обозначает лучшие
	
	setFieldCheckLikeCurField() - метод, который делает массив для проверки
	таким же как массив текущих позиций клеток
	
	getFigure() - метод, принимающий номер позиции 1 2 или 3
	и возвращает фигуру для этой позиции
	
	reWritefield() - метод переписывающий массив с полем позиций фигур 
	с полем фигуры по i j. Принимает поле, поле фигуры, i, j.
